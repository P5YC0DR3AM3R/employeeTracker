## Below is a short description:

### What was your motivation?
I was motivated to build this employee tracker as a simple tool to keep some employee data in a database.

### Why did you build this project?
It is a powerful tool for quickly referencing employee information and updating that information through answering simple prompts. It is also designed to practice using all of the things that I have learned in class.

### What problem does it solve?
It provides a single place to reference employee data and interact with the database.

### What did you learn?
postgres, SQL, inquirer
